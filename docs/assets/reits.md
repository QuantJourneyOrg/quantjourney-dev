# Module Information

**Description:**
REITs Assets class for QuantJourney Framework

The REITs class provides methods for fetching REIT data from various sources,
such as EOD Historical Data, Yahoo Finance, and OANDA.

### **Method: get_reit_data**
```python
def get_reit_data(
			symbols,
			start_date,
			end_date
		)
```
